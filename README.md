# Bike Sharing Demand Forecasting

## Project Overview
This project analyzes bike-sharing rental demand and builds a linear regression model to predict future demand using historical usage, weather, and seasonal data.

## Business Problem
A bike-sharing company needs to understand what factors influence rental demand in order to improve bike availability, staffing, and operational planning.

## Objective
The goal of this project is to identify key demand drivers and build a predictive model that can forecast bike rentals.

## Tools Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

3. Exploratory data analysis
4. Feature engineering
5. Linear regression modeling
6. Model evaluation

## Business Value
This project shows how machine learning can support operational decision-making by helping a bike-sharing company anticipate customer demand and allocate resources more effectively.
